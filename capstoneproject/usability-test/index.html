Usability Testing Results

Observations from Testing

1. Confusing Navigation: Several users weren’t sure if clicking on a name in the sidebar would open more details. Once they discovered it, they found the feature useful.

2. Popups Are Effective: Users responded positively to the popups containing detailed information. They appreciated the visuals and the contextual layout of each profile.

3. Search Needs Refinement: Some users expected the search bar to give more intelligent results, like autocomplete or related suggestions.

What Needs Improvement

The project could benefit from a more intuitive sidebar layout. It’s not obvious that items are clickable, and the risk level color indicators could be more consistent. Also, some of the mugshots were repetitive, which made the experience feel less dynamic.

User Interaction and Impact

Most users figured out how to use the basic functions after a short period. However, the initial learning curve could be smoothed out. The project did leave an impression—users described it as "interesting" and "slightly dystopian," which aligns with the intended impact.

Modifications for Better UX

We plan to improve the visual feedback on tabs and sidebar items, diversify the mugshot images more effectively, and add search enhancements. Adding a short onboarding tooltip could help first-time users too.

Update Plan

Based on feedback, the next steps include: 1) improving sidebar card styles and interactivity, 2) generating a better variety of mugshots using different image sources, and 3) refining search logic to feel smarter. These changes aim to make the experience smoother and more engaging.
