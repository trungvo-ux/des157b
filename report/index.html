Feasibility Test

Planned Features

  • Live map with user markers (status-based colors)
  • Search bar with real-time filtering
  • Status scoring system with visual indicators
  • Sidebar with name, address, timestamp, score
  • Risk level filters (All / Low / Medium / High)

Technologies Required

  • Mapbox GL JS (for interactive map)
  • JavaScript (search/filter logic, marker updates)
  • HTML/CSS (layout and styling)
  • JSON or GeoJSON (for location/status data)

Skills Assessment

  • HTML/CSS: Proficient and fully comfortable
  • JavaScript: Competent, can implement search/filter logic
  • Mapbox GL JS: Familiar, some learning needed for advanced features
  • GeoJSON: Familiar, able to generate and parse necessary data

All required technologies are within a reasonable learning curve for the project timeframe.

Team Responsibilities

This is a solo project. All responsibilities for design, coding, and research are handled by the individual. If this changes, updates will be documented.
